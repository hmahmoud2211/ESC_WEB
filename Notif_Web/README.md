# ESC Fit Club - Modest Sportswear Website

A modern e-commerce website for modest sportswear, built with HTML, CSS, and JavaScript. The website features a responsive design, shopping cart functionality, and a clean user interface.

## Project Structure

```
Notif_Web/
├── public/
│   ├── index.html
│   ├── pages/
│   │   └── cart.html
│   ├── styles/

│   │   └── cart.css
│   ├── js/
│   │   ├── shared.js
│   │   └── cart.js
│   └── images/
```

## Features

- Responsive design that works on all devices
- Shopping cart functionality with local storage
- Product catalog with featured items
- Cart management (add, remove, update quantity)
- Order summary with tax and shipping calculations
- Promo code support
- Modern UI with animations and transitions

## Setup Instructions

1. Clone the repository or download the files
2. Place your product images in the `public/images/So_photos` directory
3. Add your logo as `ESC.svg` in the `public/images` directory
4. Add your hero background image as `hero-bg.jpg` in the `public/images` directory
5. Open `public/index.html` in a web browser to view the website

## Development

The website is built using vanilla HTML, CSS, and JavaScript. No build process is required.

### CSS Structure
- `main.css`: Global styles and layout
- `cart.css`: Cart-specific styles

### JavaScript Structure
- `shared.js`: Shared state management and cart functionality
- `cart.js`: Cart page-specific functionality

## Deployment

The website is ready to be deployed to Netlify or any other static hosting service. Simply upload the contents of the `public` directory to your hosting provider.

## Browser Support

The website is compatible with all modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
